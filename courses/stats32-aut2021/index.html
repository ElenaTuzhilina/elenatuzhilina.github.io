Course Description

This short course runs for weeks 1-5 of the quarter. The goal of the course is to familiarize students with R's tools for data analysis. Lectures will be interactive with a focus on learning by example, no prior programming experience is needed. Topics covered:

  • basic data structures

  • data manipulation

  • data visualization

  • simple statistical tests

Laptops are necessary for use in class.

Textbooks

There are no required textbooks for this course. However, R for Data Science by Hadley Wickham and Garrett Grolemund (O'Reilly Media, 2017, available for free online here) is highly recommended as an aid to understanding the course material.

Assignments

Practice questions (not graded)

Constant practice is an essential component of learning programming. After each session, a small practice assignment testing your understanding of the session material will be released. I highly encourage you to do these assignments. The answers will be released a few days later.

Final project (graded)

The only graded assignment for this class is a final project. The project proposal (20%) will be due on Oct 20 (Wed) 11:59pm, and the final project itself (80%) will be due on Oct 30 (Sat) 11:59pm. More details will be given closer to the date.
